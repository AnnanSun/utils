### 📦 安装

::: code-group

```bash [pnpm]
pnpm add @iceywu/utils
```

```bash [yarn]
yarn add @iceywu/utils
```

```bash [npm]
npm install @iceywu/utils
```

:::

### 📡 `CDN`

::: code-group

```html [jsdelivr]
<!-- 提供包含所有方法的全局变量 PureUtils -->
<script src="//cdn.jsdelivr.net/npm/@iceywu/utils"></script>
```

```html [unpkg]
<!-- 提供包含所有方法的全局变量 PureUtils -->
<script src="//unpkg.com/@iceywu/utils"></script>
```

:::
